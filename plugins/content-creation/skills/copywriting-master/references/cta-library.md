## CTA Formulas

### Action + Benefit
- Start [Achieving Result]
- Get [Specific Benefit]
- Begin [Desired Outcome]
- Unlock [Value]

### First Person (Higher Engagement)
- Start My Free Trial
- Get My Free Quote
- Send Me the Guide
- Reserve My Spot
- Yes, I Want [Benefit]

### With Urgency
- Get Started Now
- Claim Before [Date]
- Start Today - Limited Spots
- Buy Now - Sale Ends [Time]

### With Risk Reversal
- Try Free for 14 Days
- Start Risk-Free
- Get Started - Cancel Anytime
- Try It - Money-Back Guarantee

### With Social Proof
- Join [Number] Users
- Start Like [Number] Others
- Join [Number] Happy Customers

## CTA Button Styles by Page Section

### Above the Fold (Hero)
- Primary: Large, high contrast, action-focused
- Secondary: Outlined or ghost button
- Example pair: Start Free Trial + Watch Demo

### Feature Section
- Contextual CTAs that relate to features
- See [Feature] in Action
- Try [Feature] Free

### Pricing Section
- Clear action per tier
- Start Free / Start Pro / Contact Sales
- Highlight recommended plan

### Testimonial Section
- Soft CTA after social proof
- Join Them or Get Similar Results

### FAQ Section
- Answer remaining questions
- Still Have Questions? Chat With Us
- Ready to Start? [Primary CTA]

### Final CTA Section
- Restate main benefit
- Include risk reversal
- Create urgency if appropriate
- Start Your Free Trial - No Credit Card Required

## CTA Microcopy (Supporting Text)

### Under Primary CTA
- No credit card required
- Free 14-day trial
- Cancel anytime
- Setup takes 2 minutes
- Join 10,000+ happy users

### Near Form Fields
- We will never share your email
- Unsubscribe anytime
- Get instant access

### After Price
- Billed annually (save 20%)
- 30-day money-back guarantee
- Includes all features

## CTAs to Avoid

### Generic (Low Converting)
- Submit
- Click Here
- Learn More (as primary)
- Sign Up
- Buy

### Unclear
- Get Started (get started with what?)
- Continue
- Next
- Go

### Friction-Creating
- Register Now (sounds like work)
- Subscribe (alone, without benefit)
- Apply (unless actually an application)

## A/B Testing Ideas

### Test These Variables
1. First person vs second person (My vs Your)
2. With vs without urgency
3. With vs without numbers
4. Action verb choices
5. Button color/contrast
6. Button size
7. Surrounding whitespace
8. Microcopy presence

### Common Winning Patterns
- Specific beats generic
- Benefit-focused beats action-only
- First person often beats second person
- Risk reversal increases clicks
- Social proof microcopy builds trust

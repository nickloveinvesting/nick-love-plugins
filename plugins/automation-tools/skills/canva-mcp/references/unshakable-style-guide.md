# Unshakable Investor Presentation Style Guide

This is the definitive visual reference for creating presentations matching The Unshakable Investor brand. Use this when prompting Canva to generate Toby Potter / Unshakable webinar-style presentations.

---

## Brand Identity Summary

**Brand:** The Unshakable Investor  
**Host:** Dr. Toby Potter  
**Tone:** Transformational, authoritative, high-stakes, dramatic  
**Feel:** Premium coaching, cinematic, refined intensity  
**Target:** High-capacity achievers seeking structure and identity alignment

---

## Exact Color Palette

| Element | Color | Hex Code | Usage |
|---------|-------|----------|-------|
| **Primary Background** | Deep Black/Charcoal | `#0D0D0D` to `#1A1A1A` | All slide backgrounds |
| **Accent Gold** | Warm Amber/Gold | `#D4A853` or `#E5A122` | Headlines emphasis, buttons, accent lines |
| **Headline Text** | Pure White | `#FFFFFF` | Primary headlines |
| **Body Text** | Off-White | `#E8E8E8` | Supporting text, bullets |
| **Negative/Warning** | Deep Red | `#CC0000` | X icons, warning elements |
| **Secondary Accent** | Burnt Orange | `#CC6600` | Gradient overlays, fire tones |
| **Muted Background** | Dark Brown | `#2D1A0A` | Subtle warmth in backgrounds |

### Background Treatment
```
ALWAYS USE:
- Solid deep black (#0D0D0D) as base
- Subtle fire/smoke texture overlay at 15-25% opacity
- Warm amber glow emanating from bottom or corners
- Curved gold/orange line patterns in corners (bottom-left and/or top-right)
```

## Typography Specifications

### Headlines
```
Font Style: Bold condensed sans-serif (Impact, Oswald Bold, or similar)
Weight: Extra Bold / Black
Case: ALL CAPS
Size: 60-80pt (dominant, takes 40-60% of slide width)
Color: White with gold accent words
Tracking: Slightly condensed
```

### Accent Words (Gold Highlights)
```
Same font as headline
Color: #D4A853 (warm gold)
Used for: Key concept words, numbers, transformational terms
Examples: "IDENTITY GAP", "SECRET #1", "OPERATING SYSTEM", "UNSTOPPABLE"
```

### Body Text
```
Font Style: Clean sans-serif (Open Sans, Lato, or similar)
Weight: Regular to Medium
Case: Sentence case
Size: 22-28pt
Color: #E8E8E8 (off-white)
Line Height: 1.4-1.5x
Max Width: 50-60% of slide for readability
```

### Subheadings/Labels
```
Font Style: Same as body, Medium weight
Size: 18-20pt
Color: White or Gold
Case: Sentence case or Title Case
```

## Layout Patterns (Slide Types)

### Type 1: Title Slide
```
LAYOUT:
- Logo: Top-left corner, ~120px width
- Headline: Left-aligned, upper-left quadrant, ALL CAPS with gold accents
- Subtitle: Below headline, smaller, sentence case
- Bullet points: Small text with gold bullet markers
- Speaker photo: Right 45% of slide, cut-out style or full-bleed
- Host credit: "YOUR HOST: DR. TOBY POTTER" in gold, bottom-left
- Background: Dark with fire/smoke texture, curved line accents

VISUAL BALANCE: 55% text (left) / 45% image (right)
```

### Type 2: Statement Slide (Bold Message)
```
LAYOUT:
- Logo: Top-left corner
- Headline: Centered or left-aligned, massive (takes 50%+ of slide)
- First line: White
- Accent line: Gold (key phrase)
- Supporting text: 2-4 lines below headline, off-white, smaller
- Background: Pure dark with subtle texture and curved line accents
- NO image on these slides

PURPOSE: Deliver one powerful message
```

### Type 3: Section Transition ("Let's Get Into...")
```
LAYOUT:
- Logo: Top-left
- Headline: "LET'S GET INTO" in white, "SECRET #1" in gold below
- Supporting text: 2-3 teaser lines on left (40%)
- Image: Right side (50%), rounded corners, relevant metaphor image
- Background: Dark with accent lines

PURPOSE: Build anticipation before key concept
```

### Type 4: Content + Image (Teaching Slides)
```
LAYOUT:
- Logo: Top-left
- Headline: Left side, bold, ALL CAPS with gold accent
- Body text: 3-5 lines below headline, left-aligned
- Image: Right 45-50%, rounded corners (~16px radius), subtle shadow
- Background: Dark with texture

VISUAL BALANCE: 50/50 split (text left, image right)
IMAGE STYLE: Professional photography, dramatic lighting, metaphorical
```

### Type 5: Problem/Pain Point Slide (Cards with X)
```
LAYOUT:
- Logo: Top-left
- Headline: Left 40%, bold statement
- Supporting text: Below headline, explains the concept
- Cards: Right 50%, stacked vertically (3 cards typical)
  - Each card: Dark semi-transparent background (#1A1A1A at 80%)
  - Gold border or outline
  - Red circle X icon on left of each card
  - White text describing the problem
- Background: Dark with fire texture

CARD STYLING:
- Rounded corners (~12px)
- Slight gold/amber border glow
- Red X: Circular background (#CC0000), white X inside
```

### Type 6: Engagement/Poll Slide
```
LAYOUT:
- Logo: Top-left
- Headline: Centered, "QUICK CHECK-IN" style
- Question: Below headline, centered, 28-32pt
- Instruction: "Type 'Yes' in the chat" style prompt
- Buttons: Two side-by-side
  - YES: Gold background (#D4A853), dark text, rounded
  - NO: Dark gray background (#4A4A4A), white text, rounded
- Background: Dark with prominent accent lines

BUTTON STYLING:
- Rounded corners (~25px, pill-like)
- Generous padding
- Bold text, ALL CAPS
```

### Type 7: List/Framework Slide
```
LAYOUT:
- Logo: Top-left
- Headline: Left, bold with gold accent
- List items: Stacked, each on its own line
- Bullet style: None (numbered) or gold squares
- Background: Dark with subtle texture

LIST FORMATTING:
"Your goals are built for your future self.
Your habits are built for your old self.
Your environment is built for your stressed self.
Your identity is built for your wounded self."

(Parallel structure, poetic rhythm)
```

## Visual Elements & Assets

### Logo Placement
```
Position: Top-left corner on EVERY slide
Size: ~100-120px width
Style: The Unshakable Investor shield logo with figure
Clearance: Minimum 40px from edges
```

### Curved Accent Lines
```
Style: Concentric curved lines radiating from corner
Color: Gold/amber (#D4A853) at 60-80% opacity
Position: Bottom-left corner AND/OR top-right corner
Purpose: Adds movement, premium feel, brand consistency
```

### Fire/Smoke Texture
```
Style: Subtle photographic fire/smoke overlay
Opacity: 15-25%
Color: Warm orange/amber tones
Position: Lower portion of slide, subtle glow effect
Purpose: Transformation metaphor, energy, warmth against dark
```

### Image Treatment
```
Shape: Rounded corners (~16px radius)
Shadow: Subtle drop shadow (8px blur, 20% opacity, dark)
Border: Optional thin gold border for emphasis
Style: High-quality photography, dramatic lighting
Subjects: People in contemplation, transformation imagery, metaphors
```

### Icons
```
Style: Simple, modern
Negative items: Red circle with white X
Positive items: Gold checkmark (if used)
Size: 40-50px
```

## Query Template for Canva

Use this structure when prompting for Unshakable-style presentations:

```
**Style Guide:**
OVERALL: Dramatic transformational coaching style. Cinematic dark mode with fire/warmth undertones.

COLORS:
- Background: Deep black (#0D0D0D) with subtle fire/smoke texture overlay
- Primary Accent: Warm amber gold (#D4A853) for emphasis words and accent lines
- Text: White headlines, off-white body (#E8E8E8)
- Warning/negative: Deep red (#CC0000) for X icons

TYPOGRAPHY:
- Headlines: Bold condensed ALL CAPS sans-serif, 60-80pt, white with gold accent words
- Body: Clean sans-serif, 22-28pt, off-white, left-aligned
- Key phrases in gold for emphasis

VISUAL ELEMENTS:
- Unshakable Investor logo: Top-left corner on all slides, 100px width
- Curved gold line accents: Bottom-left and top-right corners
- Subtle fire/smoke texture in backgrounds at 20% opacity
- Images: Rounded corners (16px), dramatic lighting, right side of split layouts

LAYOUT PATTERNS:
- Title slide: 55/45 text-left, speaker-image-right split
- Teaching slides: 50/50 content-left, image-right split
- Statement slides: Bold centered/left headline, minimal supporting text
- Problem slides: Headline left, stacked cards with red X icons on right

IMAGE STYLE:
- Professional photography with dramatic, moody lighting
- Metaphorical imagery (mirrors, sculptures, mountains, fire)
- Cut-out speaker photos with transparent backgrounds
- Rounded corners on all contained images

CONSISTENCY:
□ Logo top-left on every slide
□ Curved gold accent lines in corners
□ Fire/smoke texture throughout
□ Gold highlighting on key concept words
□ Parallel sentence structure in lists
□ One core message per slide
```

## Anti-Patterns (What NOT to Do)

❌ **Don't use** bright/light backgrounds  
❌ **Don't use** more than 2 fonts  
❌ **Don't use** square-cornered images  
❌ **Don't use** blue or cool colors as accents  
❌ **Don't use** stock photography with bright/happy lighting  
❌ **Don't use** bullet points with circles (use gold squares or none)  
❌ **Don't use** clip art or cartoon illustrations  
❌ **Don't use** gradients that aren't fire/warmth tones  
❌ **Don't crowd** slides with too much text (one message per slide)  
❌ **Don't forget** the curved accent lines in corners  
❌ **Don't forget** to highlight key words in gold

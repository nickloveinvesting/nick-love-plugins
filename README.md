# Nick Love Plugins

Nick Love's personal plugin marketplace for Claude Code and Cowork — 33 skills organized across 6 plugins.

## Add this marketplace

```
/plugin marketplace add nickloveinvesting/nick-love-plugins
```

## Available Plugins

| Plugin | Skills | Description |
|--------|--------|-------------|
| `content-creation` | 5 | Copywriting, email, carousels, landing page copy, research |
| `design-tools` | 7 | AI image prompting, logo, canvas art, design system, Excalidraw |
| `document-tools` | 4 | Word docs, PDFs, PowerPoint, Excel |
| `web-dev` | 7 | Frontend dev, landing pages, Playwright testing, UI auditing |
| `automation-tools` | 6 | Make.com, Chrome, Canva, scheduler, NotebookLM, Facebook Ads |
| `meta-skills` | 4 | Skill builder, plugin discovery, recursive problem solver |

## Install a plugin

```
/plugin install content-creation@nick-love-plugins
/plugin install design-tools@nick-love-plugins
/plugin install document-tools@nick-love-plugins
/plugin install web-dev@nick-love-plugins
/plugin install automation-tools@nick-love-plugins
/plugin install meta-skills@nick-love-plugins
```

## Update

```
/plugin marketplace update nick-love-plugins
```

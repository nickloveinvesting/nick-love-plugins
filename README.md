# Nick Love Plugins

Nick Love's personal plugin marketplace for Claude Code and Cowork — 105 plugins across content, design, documents, web dev, automation, AI/ML, data engineering, and more.

## Add this marketplace

```
/plugin marketplace add nickloveinvesting/nick-love-plugins
```

## Install a plugin

```
/plugin install <plugin-name>@nick-love-plugins
```

## Update

```
/plugin marketplace update nick-love-plugins
```
